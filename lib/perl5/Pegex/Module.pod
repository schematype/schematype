=head1 NAME

Pegex::Module - Base Class for Pegex Grammar Interface Modules

=head1 SYNOPSIS

    package MyLanguage;
    use Pegex::Base;
    extends 'Pegex::Module';

    use Pegex::Parser;

    has parser_class => 'Pegex::Parser';
    has grammar_class => 'MyLanguage::Grammar';
    has receiver_class => 'MyLanguage::AST';

    1;

=head1 DESCRIPTION

The module in the Synopsis above is a complete language parsing module. It
just inherits from L<Pegex::Module>, and then overrides the
C<grammar_class> and C<receiver_class> attributes. L<Pegex::Module>
provides the C<parse()> method.
